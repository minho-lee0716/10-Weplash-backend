# Unsplash Clone Project
***
* Cloning a website dedicated to sharing stock photography under the Unsplash license.

### 개발 인원 및 기간
* 개발 기간: 2020/08/03 ~ 2020/08/14
* 개발 인원: 프론트엔드 3명, 백엔드 3명

## 팀원
* Front-End : 이병수, 이지윤, 임현지
* Back-End : 손수정, 이민호, 김동현
